# 30 Days of UX Principles

An Android application that presents a daily dose of User Experience (UX) principles to help developers and designers build better products. This project is built using modern Android development practices, including **Jetpack Compose**.

## Features

- **30 Daily Tips**: A curated list of 30 essential UX principles, from Visual Hierarchy to Gestalt Principles.
- **Detailed Descriptions**: Each tip comes with a clear explanation and its importance in design.
- **Visual Learning**: Every principle is accompanied by an illustrative image to aid understanding.
- **Welcome Screen**: A beautiful onboarding experience to introduce users to the app.
- **Modern UI**: Built entirely with Jetpack Compose for a smooth, responsive, and maintainable interface.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Build System**: Gradle (Kotlin DSL)
- **Architecture**: Simple Model-Repository pattern for data handling.

## Getting Started

1.  Clone the repository.
2.  Open the project in Android Studio (Ladybug or newer recommended).
3.  Build and run the app on an emulator or a physical device.

## UX Principles Covered

Some of the principles included in this app:
- Visual Hierarchy
- Contrast & Typography
- Color Theory
- Consistency & User Feedback
- Accessibility
- Fitts's Law & Hick's Law
- Gestalt Principles (Proximity, Similarity, etc.)
- And many more!
